# OMNIDOM — Knowledge. Services. Growth.

OMNIDOM is a lightweight, responsive, single-page web platform and content management system (CMS) designed for content creators, agencies, and independent service providers. Built with a modern, clean UI, it brings blogs, service catalogs, digital products, and lead generation together under one roof, backed by a real-time serverless database.

## 🚀 Features

### Public Site
- **Dynamic Blog Engine:** Read full-length articles with search filters and clean typography.
- **Service Catalog:** Display premium service offerings with instant "Get in Touch" lead capture.
- **Resource Tab:** Segment and distribute educational digital assets like E-Books and Video Courses.
- **Responsive Navigation & UI:** Fast, mobile-first design leveraging Google Fonts (`Plus Jakarta Sans` & `Instrument Serif`) and fluent custom CSS variables.

### Admin Dashboard (Control Panel)
- **Firebase Authentication:** Secure login gateway using modern authentication handlers.
- **Real-Time Data Layer:** Perform full CRUD operations on blogs, services, products, and contact leads.
- **Rich Text Editor:** Built-in semantic toolbar to write and publish blogs effortlessly.
- **Analytics Metrics:** At-a-glance visualization tracking total blog posts, services listed, active digital products, and incoming business leads.

---

## 🛠️ Tech Stack

- **Frontend:** Vanilla HTML5, CSS3 Custom Properties (Variables), modern JavaScript Modules (ES6+).
- **Database & Backend:** Google Firebase (Firestore Serverless DB).
- **Authentication:** Firebase Auth.

Since this site is built purely using frontend modules and serverless logic, it requires no traditional build step (`npm run dev`). You can run it locally out of the box using any standard local server wrapper (such as VS Code's *Live Server* or Python's `http.server`).
